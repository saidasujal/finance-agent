>_ Developer CLI Console Guide

NexFi features a built-in terminal for power users and developers. Execute instant actions, manage budget parameters, and inspect application status without touching a mouse. Use the toggle button above to open or close the drawer console.

/add [amount] [category] [description]
Logs a new expense transaction. e.g., /add 250 Food lunch
/goal [amount]
Updates the target monthly budget goal. e.g., /goal 8000
/help
Prints a list of all supported interactive terminal commands.
/clear & /exit
Use /clear to flush console output or /exit to close.

🛡️ Security & Privacy

  • Localized Database: All expenses are recorded on your machine in a secure local SQLite DB instance.
  • TLS Encryption: Interactions with the Google Gemini API are fully encrypted in transit.
  • No Third-Party Analytics: Zero external tracking scripts, cookies, or data brokers involved.
